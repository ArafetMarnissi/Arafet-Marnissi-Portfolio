# Hackathon Photos

Add your hackathon photos here with the following naming convention:

- hack1.jpg
- hack2.jpg
- hack3.jpg
- etc.

Recommended image specifications:

- Aspect ratio: 16:10 (landscape)
- Resolution: 1280x800 or higher
- Format: JPG or PNG
- File size: < 2MB each

The photos will be displayed in a horizontal scrollable gallery in the Hero section.

## Required Images:

1. **climate-change.jpg** - For "AI for Climate Change" hackathon

   - Suggested: Environmental data visualization, renewable energy graphics, climate tech

2. **medical-analysis.jpg** - For "Medical Image Analysis" hackathon

   - Suggested: Medical imaging, skin cancer detection interface, healthcare tech

3. **smart-transport.jpg** - For "Smart Transportation" hackathon

   - Suggested: Traffic flow visualization, smart city transport, urban mobility

4. **fintech-ai.jpg** - For "Financial AI Assistant" hackathon
   - Suggested: Financial dashboard, investment charts, fintech interface

## Specifications:

- **Format**: JPG, PNG, or WebP
- **Size**: 800x600 pixels (4:3 aspect ratio recommended)
- **File size**: Under 500KB each for optimal loading
- **Quality**: High resolution, clear and professional

## Tips:

- Use screenshots of your actual hackathon projects if available
- Consider relevant stock images or concept visualizations
- Make sure images relate to the specific hackathon theme
- Professional appearance that reflects the award-winning nature

The images will be displayed as:

- 192px height (h-48)
- Full width of the card
- Rounded top corners
- Hover effect with scale and overlay
