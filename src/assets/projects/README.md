# Projects Images

Add your project images to this folder. Here are the expected filenames:

## Required Images:

1. **chatbot.jpg** - For "AI-Powered Chatbot" project

   - Suggested: Screenshot of chatbot interface, AI conversation, or bot icon

2. **computer-vision.jpg** - For "Computer Vision Analytics" project

   - Suggested: Object detection visualization, car/traffic analysis, YOLO detection boxes

3. **analytics-dashboard.jpg** - For "Predictive Analytics Dashboard" project

   - Suggested: Dashboard screenshot, charts/graphs, data visualization

4. **style-transfer.jpg** - For "Neural Style Transfer App" project
   - Suggested: Before/after style transfer examples, artistic neural network visualization

## Specifications:

- **Format**: JPG, PNG, or WebP
- **Size**: 800x600 pixels (4:3 aspect ratio recommended)
- **File size**: Under 500KB each for optimal loading
- **Quality**: High resolution, clear and professional

## Tips:

- Use screenshots of your actual projects if available
- Consider mockups or concept images if projects are not visual
- Ensure images are relevant to the technology stack mentioned
- Professional and clean appearance works best

The images will be displayed as:

- 192px height (h-48)
- Full width of the card
- Rounded top corners
- Hover effect with scale and overlay
